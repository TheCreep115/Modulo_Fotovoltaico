 ☀️ Photovoltaic Monitoring System (ESP32 + Web)

 📌 Description

This project is a web-based monitoring and control system for a photovoltaic module powered by an ESP32. It provides real-time visualization of key electrical parameters such as voltage, current, and power, while also enabling automated panel orientation and battery management.

The system integrates hardware, a backend, and a web interface to deliver an efficient and smart solar energy solution.

---

 🚀 Features

* 📊 Real-time monitoring of:

  * Voltage
  * Current
  * Power output
* 🔄 Automated panel orientation using actuators
* 🔋 Battery charge management
* 🌐 Web interface for data visualization
* 📡 ESP32 WiFi communication with database
* 🗄️ Data storage using MySQL

---

 🛠️ Technologies Used

🔹 Hardware

 ESP32
* Voltage and current sensors
* Photovoltaic module
* Actuators (pistons/motors)

 🔹 Software

* Frontend: HTML, CSS, JavaScript
* Backend: (e.g., PHP / Spring Boot if applies)
* Database: MySQL
* Communication: HTTP / WiFi

---

 ⚙️ System Architecture

1. The ESP32 collects data from sensors
2. Data is sent via WiFi to a server/database
3. The backend processes and stores the data
4. The web interface displays real-time information
5. The system can control actuators for panel orientation
